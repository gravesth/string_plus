#### Функции для организации структуры (Suite / TCase / Runner)

| Функция                                            | Назначение                                        |
| -------------------------------------------------- | ------------------------------------------------- |
| `Suite *suite_create(char *name)`                  | Создает набор тестов (группирует TCase).          |
| `TCase *tcase_create(char *name)`                  | Создает тестовый случай (группирует тесты).       |
| `void suite_add_tcase(Suite *s, TCase *tc)`        | Добавляет TCase в Suite.                          |
| `void tcase_add_test(TCase *tc, TCaseTestFn test)` | Добавляет конкретную тест-функцию в TCase         |
| `SRunner *srunner_create(Suite *s)`                | Создает "бегунок", который будет исполнять тесты. |


#### Управление запуском

| Функция                                  | Назначение                                                        |
| ---------------------------------------- | ----------------------------------------------------------------- |
| `srunner_run_all(sr, CK_VERBOSE)`        | Запускает тесты (второй аргумент — режим вывода).                 |
| `srunner_ntests_failed(sr)`              | Возвращает количество упавших тестов (используется для `return`). |
| `srunner_free(sr)`                       | Освобождает память бегунка.                                       |
| `srunner_set_fork_status(sr, CK_NOFORK)` | Отключает `fork` (удобно для отладки в IDE)                       |
