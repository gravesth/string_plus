## Синтаксис
```
void *memchr(const void *str, int c, size_t n);
```

- **`str`** - указатель на блок памяти (массив байт), где происходит поиск
- **`c`** - искомый символ (передается как `int`, но используется как `unsigned char`)
- **`n`** - количество байт, в которых нужно искать

## Алгоритм работы
1. **Приведение типов**: Функция интерпретирует `c` как `unsigned char` (значение от 0 до 255)
2. **Побайтовый поиск**: Последовательно проверяет каждый байт в диапазоне от `str` до `str + n - 1`
3. **Сравнение**: Сравнивает текущий байт с преобразованным значением `c`
4. **Результат**:
    - Если символ найден → возвращает указатель на его позицию
    - Если не найден за `n` байт → возвращает `NULL`
